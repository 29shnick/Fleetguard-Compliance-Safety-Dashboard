# 🚛 Fleet Compliance & Safety Audit Dashboard
**Bridging Logistics Operations & Data Analytics**

## 🎯 Executive Summary
As a former Safety and Operations Manager, I identified a critical need for real-time compliance monitoring in the trucking industry. This project transitions traditional "reactive" safety management into a "proactive" IT solution. It automates the tracking of DOT-mandated expirations to mitigate risk and ensure fleet uptime.

## 🛠 Tech Stack
* **Language:** Python 3.x
* **Data Engine:** Pandas
* **Deployment:** Streamlit Cloud
* **Version Control:** GitHub

## 📈 Project Management & Business Impact
This dashboard demonstrates the following core PM competencies:

* **Risk Mitigation:** Implemented logic to trigger "Warning" statuses 30 days prior to expiration, allowing for scheduled maintenance and physicals before a violation occurs.
* **Stakeholder Reporting:** Developed a "Fleet Health Score" (KPI) to provide executives with an immediate high-level overview of fleet legality.
* **Operational Efficiency:** Reduced the manual audit time by aggregating driver CDL data and vehicle inspection records into a single, filterable source of truth.
* **Quality Assurance:** Developed a mock dataset to validate compliance logic across "Expired," "Warning," and "Compliant" scenarios.
